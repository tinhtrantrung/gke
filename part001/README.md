# Kubernetes - Part One

To run this code here are the quick steps.

### To startup and deploy:
```bash
git clone https://github.com/tinhtrantrung/k8s.git
cd k8s/part001/scripts
sh startup.sh
sh deploy.sh
sh check-endpoint.sh part001
```

### To Teardown:
```bash
cd k8s/part001/scripts # if necessary
sh teardown.sh
```
